# Support

## Getting Help

- **Bug Reports** — Use the [bug report template](../../issues/new?template=bug_report.yml) to report issues
- **Feature Requests** — Use the [feature request template](../../issues/new?template=feature_request.yml)
- **Security Issues** — See [SECURITY.md](SECURITY.md) for responsible disclosure

## Response Times

| Type | Target Response |
|------|----------------|
| Security vulnerabilities | 48 hours |
| Bug reports | 1 week |
| Feature requests | 2 weeks |

## Before Opening an Issue

1. Search existing issues to avoid duplicates
2. Include reproduction steps for bugs
3. Provide environment details (OS, browser, app version)
